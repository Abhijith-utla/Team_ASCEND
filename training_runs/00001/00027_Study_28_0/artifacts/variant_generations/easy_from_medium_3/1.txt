Here are the three simpler versions of the sat() function:

VARIANT1:
```python
def sat(li):
    for i in li:
        for j in li:
            if abs(i - j) < 10:
                return False
    return len(li) == 100
```

VARIANT2:
```python
def sat(li):
    return not any(abs(i - j) < 10 for i in li for j in li if i != j) and len(li) == 100
```

VARIANT3:
```python
def sat(li):
    return all(abs(i - j) >= 10 for i in li for j in li if i != j) and len(li) == 100
```
