```python
from typing import List

def sat(li: List[int]):
    return all(j in {i - 1, i + 1, 3 * i} for i, j in zip([0] + li, li + [128]))
```

This function takes a list of integers as input and returns True if every element in the list is in the set {i - 1, i + 1, 3 * i} for some integer i and j, where j is the i-th element in the list. If the list is empty, the function will return True.
