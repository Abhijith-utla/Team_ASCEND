```python
from typing import List

def sat(li: List[int]) -> bool:
    for i, j in enumerate(li):
        if j != 3 * i:
            return False
    return True
```

The function uses the `enumerate` function to get the index and value of each element in the list. If any element is not a multiple of 3, it immediately returns False.
