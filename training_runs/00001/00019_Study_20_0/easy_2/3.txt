```python
from typing import List

def sol(li: List[int]) -> int:
    for i in range(1, len(li) + 1):
        if i not in {li[j - 1] - 1 for j in range(1, len(li) + 1)} and \
           i not in {li[j + 1] + 1 for j in range(1, len(li) + 1)} and \
           i not in {3 * li[j - 1] for j in range(1, len(li) + 1)} and \
           i not in {3 * li[j + 1] for j in range(1, len(li) + 1)}:
            return i
    return None
```

This function first checks if each element in the list is either 1, 2, or 3 times their position. If the condition is not met for any element, the function returns the position of the first element that fails the condition. If no such element is found, the function returns None.
