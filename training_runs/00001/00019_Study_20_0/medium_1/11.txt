```python
def sol():
    def sat(li: List[int]):
        return all((j in {i - 1, i + 1, 3 * i} for i, j in zip([0] + li, li + [128])))

    return sat
```

In the above code, `sol()` is a function that returns the `sat` function. The `sat` function is a nested function that checks if each number in the list is in the set of numbers {i - 1, i + 1, 3 * i} where i is the index of the number in the list.
