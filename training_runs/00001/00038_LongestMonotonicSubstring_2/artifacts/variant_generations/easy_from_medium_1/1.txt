Here are three variations of the function:

VARIANT1:
```python
def sat(x):
    return all(x[i] <= x[i + 1] and x[i + 1] > x[i] >= 0 for i in range(len(x) - 1))
```

VARIANT2:
```python
def sat(x):
    return all(x[i] <= x[i + 1] and x[i + 1] > x[i] for i in range(len(x) - 1))
```

VARIANT3:
```python
def sat(x):
    return all(x[i] <= x[i + 1] for i in range(len(x) - 1))
```
