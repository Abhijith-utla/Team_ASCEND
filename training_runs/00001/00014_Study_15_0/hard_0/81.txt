```python
def sat(li: List[int]):
    return all(sum(li[:i]) == 2 ** i - 1 for i in range(20))
```

This function checks if the list `li` fulfills the condition `sum(li[:i]) == 2 ** i - 1` for `i` from `0` to `19`. It returns `True` if all conditions are met, `False` otherwise.
