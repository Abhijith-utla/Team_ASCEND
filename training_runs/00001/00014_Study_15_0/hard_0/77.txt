```python
from typing import List

def sat(li: List[int]):
    return all(sum(li[:i]) == 2 ** i - 1 for i in range(20))
```

In this function, we use Python's built-in function `all` to check if all the elements in the list satisfy the condition given by the sum of the elements up to the current index `i` is equal to `2**i - 1`. This condition is true for all indices from `0` to `19`. If all elements in the list satisfy this condition, `all` returns `True`, otherwise it returns `False`.
