Here is a Python solution for the problem. This solution assumes that the list only contains positive integers and the sum of all elements in the list is less than or equal to 2^20 - 1.

```python
from typing import List

def sat(li: List[int]) -> bool:
    if not all(1 <= x <= 20 for x in li):
        return False
    if sum(li) > 2**20 - 1:
        return False
    return all(sum(li[:i]) == 2 ** i - 1 for i in range(20))
```

The function `sat` first checks if all elements in the list are within the range of 1 to 20 (inclusive). If not, it immediately returns False. Then it checks if the sum of all elements in the list is greater than 2^20 - 1. If it is, it returns False. Finally, it returns a Boolean value indicating whether all elements in the list satisfy the condition that the sum of all elements up to that point is equal to 2^i - 1 for i in the range 0 to 19.
