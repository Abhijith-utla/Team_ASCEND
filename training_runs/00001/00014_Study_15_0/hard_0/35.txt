```python
from typing import List

def sat(li: List[int]) -> bool:
    return all(sum(li[:i]) == 2 ** i - 1 for i in range(20))
```

This code checks if the sum of all elements in the list is equal to 2 raised to the power of the index minus 1 for all indices from 0 to 19. If all these conditions are met, it returns True, otherwise it returns False.
