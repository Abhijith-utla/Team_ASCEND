Here's a Python solution for the problem. This solution uses list comprehension to generate a list of results, where each result is True if the sum of the first i elements in the list is equal to 2 ** i - 1 for i in the range from 0 to 19. The all function then returns True if all the results are True, and False otherwise.

```python
from typing import List

def sat(li: List[int]) -> bool:
    return all(sum(li[:i]) == 2 ** i - 1 for i in range(20))
```
