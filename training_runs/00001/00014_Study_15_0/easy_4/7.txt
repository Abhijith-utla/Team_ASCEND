```python
from typing import List

def sat(li: List[int]):
    for x in range(20):
        for y in li[:x]:
            if x + y != 2 ** x:
                return False
    return True
```

The function `sat` checks if every sum `x + y` for all `x` from 0 to 19 exists in the list `li`. It iterates over all possible values of `x` and for each `x`, it iterates over all elements of `li` up to the current index `x`. If the sum `x + y` is not equal to `2 ** x`, the function returns `False`. If it finishes the iteration without finding any discrepancies, it returns `True`.
