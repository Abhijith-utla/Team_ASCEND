```python
def sat(s: str):
    return 'oo' in s and s.count('o') == 1000

def sol():
    answer = sat('oo' * 1000)
    return answer

print(sol())
```

The function `sol` calls `sat` with a string `'oo' * 1000` as an argument and returns the result.

The `sol` function is called at the end and the result is printed.
