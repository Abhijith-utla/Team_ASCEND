Here is the Python code that satisfies the given conditions:

```python
def sat(s: str):
    return s.count('o') == 1000 and s.count('oo') == 0
```

This function checks if the string `s` contains exactly 1000 'o's and no 'oo's. If both conditions are met, the function returns `True`, otherwise it returns `False`.

You can use this function by calling it with a string as an argument. Here's an example:

```python
print(sat("oo" * 1000))  # Output: True
print(sat("oo" * 999 + "o"))  # Output: False
```

In the first example, the function will return `True` because the string contains 1000 'o's and no 'oo's.

In the second example, the function will return `False` because the string contains 999 'o's and 1 'oo' because the 'oo' is the last character.
