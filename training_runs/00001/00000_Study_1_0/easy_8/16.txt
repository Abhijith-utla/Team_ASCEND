Here's a Python solution for the problem:

```python
def sat(s: str):
    return 'o' not in s and 'oo' not in s and s.count('o') == 1000
```

This function `sat` checks if the string `s` contains exactly 1000 'o' characters and neither 'oo' (double 'o') occurs in the string. If all conditions are met, the function returns `True`, otherwise `False`. The function takes a string argument `s` and returns a boolean value.

Here is the usage of the function:

```python
print(sat("oooooooooo"))  # True
print(sat("ooooooooooo"))  # False
print(sat("oooooooo"))  # True
print(sat("ooooooooo"))  # False
print(sat("ooooooooooo"))  # False
```
