The Python function 'sat' checks if the string 's' contains exactly one 'o' character. It also checks if there are no 'oo' characters in 's' and if the total number of 'o' characters in 's' is 1000. 

Here is the correct implementation of the function:

```python
def sat(s: str):
    return 'o' in s and 'oo' not in s and s.count('o') == 1000
```

This function will return True if the string 's' satisfies all the conditions. If not, it will return False. The function will return True if the string 's' consists of exactly one 'o' character and there are no 'oo' characters in 's'. Also, if the total number of 'o' characters in 's' is 1000.
