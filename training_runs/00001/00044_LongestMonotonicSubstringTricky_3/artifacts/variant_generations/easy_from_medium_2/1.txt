VARIANT1:
```python
def sat(x: int, y: int) -> bool:
    return x <= y
```

VARIANT2:
```python
def sat(x: List[int], y: List[int]) -> bool:
    return all(x[i] <= y[i] for i in range(len(x)))
```

VARIANT3:
```python
def sat(x: List[int], y: List[int]) -> bool:
    return all(x[i] <= x[i + 1] and x[i + 1] > x[i] for i in range(len(x) - 1))
```
