The correct function should use the length of the string to determine the length of the set of characters. Here is the correct version:

```python
from typing import List

def sat(ls: List[str]):
    if len(ls) == 0:
        return True
    else:
        return all(len(ls[0]) == len(i) for i in ls)
```
