The function `sat` takes a list of strings and checks if all strings in the list are equal in length and the maximum and minimum of the list are the same. If all these conditions are met, the function returns `True`, otherwise it returns `False`.

Here is the Python code that implements this function:

```python
from typing import List

def sat(ls: List[str]):
    return min(ls) == max(ls) == str(len(ls))
```

You can use this function to check if a list of strings satisfies the conditions of the problem. For example,

```python
print(sat(["a", "a", "a"]))  # Output: True
print(sat(["a", "b", "c"]))  # Output: False
```
