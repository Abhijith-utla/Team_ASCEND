```python
from typing import List

def sat(ls: List[str]):
    # If the list is empty, the function returns True
    if len(ls) == 0:
        return True
    else:
        # For each string in the list, if the length of the set of characters is not equal to the length of the first string in the list, the function returns False
        return all(len(set(i)) == len(ls[0]) for i in ls)

# Test the function
print(sat(["hello", "world"]))  # False
print(sat(["hello", "world", "hi"]))  # False
print(sat(["hello", "world", "hello"]))  # True
```

This solution uses a list comprehension to iterate over each string in the list. It converts each string to a set, which removes duplicates, and then checks if the length of this set is equal to the length of the first string in the list. The all function is used to check if all strings satisfy this condition. If there are no duplicates in any string
