Otherwise, it checks if all strings in the list have the same number of unique characters by converting each string to a set (which removes any duplicate characters) and comparing the length of the set to the length of the first string in the list. If all strings have the same number of unique characters, the function returns `True`; otherwise, it returns `False`.

Here is the Python code for the function `sol`:

```python
from typing import List

def sol(ls: List[str]):
    if len(ls) == 0:
        return True
    else:
        return all(len(set(i)) == len(ls[0]) for i in ls)
```
