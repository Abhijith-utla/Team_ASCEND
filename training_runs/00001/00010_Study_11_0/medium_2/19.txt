```python
from typing import List

def sat(ls: List[str]):
    if len(ls) == 0:
        return True
    else:
        return all(len(i) == len(ls[0]) for i in ls)

# Test
print(sat([]))  # True
print(sat(["abc", "def", "ghi"]))  # False
print(sat(["abc", "def", "ghij"]))  # True
```

In this solution, the function `sat` is checking if the list `ls` is empty or all elements in the list have the same length. The `all` function is used to check if all elements in the generator expression `(len(i) == len(ls[0]) for i in ls)` are True. If all elements are True, `all` returns True; otherwise, it returns False. If `ls` is empty, `all` returns True by default, so `sat` returns True when the list is empty.
