```python
from typing import List

def sat(ls: List[str]):
    if len(ls) == 0:
        return True
    else:
        return all(len(set(i)) == len(ls[0]) for i in ls)

# Test the function
assert sat(['abc', 'def', 'ghi'])
assert not sat(['abc', 'bcd', 'ghi'])
```

The function `sat` checks if all strings in a list have the same length. It uses a set to remove duplicates from each string and then checks if the length of the set is equal to the length of the first string in the list. The `all` function is used to check if all strings in the list meet this condition, and `assert` is used to validate the solution.
