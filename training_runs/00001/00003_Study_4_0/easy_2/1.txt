The function `sat` is a checker function that checks if the input list `li` contains exactly one instance of the number 3. If the list `li` contains more or less than 10 elements, the function returns `False`. If the count of the number 3 in the list is not exactly 2, the function also returns `False`. Otherwise, the function returns `True`.

```python
from typing import List

def sat(li: List[int]) -> bool:
    if len(li) != 10:
        return False
    if li.count(li[3]) != 2:
        return False
    return True
```
