The function `sat` is already defined by the user and it checks if the list has exactly 10 elements. If not, it returns `False`, otherwise it returns `True`. 

Here's the Python code for it:

```python
from typing import List

def sat(li: List[int]) -> bool:
    if len(li) != 10:
        return False
    return True
```
