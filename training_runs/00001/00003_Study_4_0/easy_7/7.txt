```python
from typing import List

def sol():
    li = [1, 2, 3, 3, 4, 5, 6, 7, 8, 9]
    return li

def sat(li: List[int]) -> bool:
    if len(li) != 10:
        return False
    if li.count(li[3]) != 2:
        return False
    return True

print(sat(sol()))
```

This will print "True" if the function sol() returns a list of 10 numbers and exactly two occurrences of the number 3 in this list, and "False" otherwise
