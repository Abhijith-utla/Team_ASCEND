Here is a Python solution for your problem. This solution uses list comprehension and the zip function to iterate over pairs of elements from the list. The sum of the pair `(i, j)` is checked to be equal to `9` and the length of the list is checked to be `1000`.

```python
from typing import List

def sat(li: List[int]):
    return all(i + j == 9 for i, j in zip([4] + li, li)) and len(li) == 1000
```
