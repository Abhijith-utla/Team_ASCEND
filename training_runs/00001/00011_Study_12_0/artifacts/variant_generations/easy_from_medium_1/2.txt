Below are the three simple variants of the given Python function named 'sat':

VARIANT1:
```python
def sat(li: List[int]):
    return all(i + j == 1 for i, j in zip([4] + li, li)) and len(li) == 1000
```

In each variant, the function checks if the sum of each element in the list is 1, 2, or 3. Additionally, it checks that the list contains exactly 1000 elements.
