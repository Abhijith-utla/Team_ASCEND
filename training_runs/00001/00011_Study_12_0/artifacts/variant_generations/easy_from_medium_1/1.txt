Here are three simpler versions of the same function that test the same concept:

```python
def sat(li: List[int]):
    return all(i == j for i, j in zip(li, [1] + li)) and len(li) == 1000

def sat(li: List[int]):
    return all(i + j == 1 for i, j in zip([4] + li, li)) and len(li) == 1000

def sat(li: List[int]):
    return all(i - j == -1 for i, j in zip(li, [0] + li)) and len(li) == 1000
```
