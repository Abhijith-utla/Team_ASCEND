```python
from typing import List

def sat(li: List[int]):
    return all([li.count(i) == i for i in range(10)]) and li.count(0) == 1
```

This function checks if the list `li` contains exactly one 0 and all numbers from 0 to 9 (inclusive) appear exactly once in `li`. The function `all()` function returns True if all elements of an iterable (in this case, a list comprehension) are True. If not, it returns False.
