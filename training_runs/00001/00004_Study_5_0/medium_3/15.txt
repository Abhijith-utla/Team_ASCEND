Here's a solution that meets the requirements:

```python
from typing import List

def sol():
    def sat(li: List[int]):
        return all([li.count(i) == i for i in range(10)]) and li.count(0) == 1

    return sat
```

This function `sol` returns another function `sat` which checks if a list of integers contains exactly one occurrence of each number from 0 to 9.

