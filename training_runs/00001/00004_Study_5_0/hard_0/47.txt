```python
from typing import List

def sat(li: List[int]):
    return all([li.count(i) == i for i in range(10)])
```

It does this by using the `count` method of the list to count how many times each number appears in the list. Then it checks if all counts are equal to the number itself, which would mean that all numbers in the list are the same.
