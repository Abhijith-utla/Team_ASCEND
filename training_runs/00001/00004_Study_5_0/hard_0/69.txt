```python
from typing import List

def sat(li: List[int]):
    for i in range(10):
        if li.count(i) != i:
            return False
    return True
```

This function `sat` takes a list of integers (`li`) as input. It then checks if all elements in the list are present exactly once. If any element is not present exactly once, the function immediately returns `False`. If all elements are present exactly once, the function returns `True`.

The `count` method of the list object counts the number of occurrences of a specified value in the list.

The `range` function generates a sequence of numbers from 0 to 9 (inclusive). 

The `for` loop then iterates over this sequence, for each `i` in this range, it checks if the count of `i` in the list `li` is exactly `i`. If either of these conditions is not met, the function immediately returns `False`, signaling that the input list does not meet the requirements.
