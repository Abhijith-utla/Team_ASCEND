def sat(li: List[int]):

Here is how it works:

- The function `count` is used to count the number of occurrences of each integer in the list `li`.
- The list comprehension `[li.count(i) == i for i in range(10)]` generates a list of boolean values. Each boolean value is `True` if the number `i` occurs the same number of times in `li` and `False` otherwise.
- The function `all` returns `True` if all the boolean values in the list are `True`, and `False` otherwise.
