The function `sat` is defined to check if all elements in the list `li` are equal to each number from 0 to 9. Here is how you could write the `sol` function to satisfy this requirement.

```python
from typing import List

def sat(li: List[int]):
    for i in range(10):
        if li.count(i) != i:

    return True
```
