```python
from typing import List

def sat(li: List[int]):
    for i in range(10):
        if li.count(i) != i:
            return False
    return True
```

The function uses the count method of the list to count the occurrence of each number in the list. If the count is not equal to the number itself, the function returns False. If it goes through the entire loop without finding such a case, it returns True, which means all elements are present exactly once.
