```python
from typing import List

def sat(li: List[int]):
    for i in range(1, 11):
        if li.count(i) != i:
            return False
    return True
```
This function checks if every number from 1 to 10 appears the same number of times in the list. If all numbers are present and the count is equal to the number, it returns True, otherwise it returns False. The function uses the count method of Python lists to count the occurrences of each number in the list. If any number is found more times than it appears in the list, the function returns False, otherwise it returns True.
