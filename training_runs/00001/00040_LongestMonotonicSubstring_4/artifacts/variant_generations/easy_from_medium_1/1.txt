```python
def sat(x):
    if len(x) < 2:
        return True
    return all(x[i] <= x[i + 1] for i in range(len(x) - 1))
```

VARIANT2:
```python
def sat(x):
    return all(x[i] <= x[i + 1] for i in range(len(x))) if len(x) > 0 else False
```

```python
def sat(x):
```
