Here is a solution in Python:

```python
from typing import List

def sat(ls: List[str]) -> bool:
    return all(len(set(s)) == len(s) for s in ls)
```

This function `sat` checks if all the strings in the input list are unique by comparing the length of the set of characters in each string and the length of the original string. If all strings are unique, the function returns `True`; otherwise, it returns `False`.
