```python
from typing import List

def sat(ls: List[str]) -> bool:
    return all(len(set(s)) == len(s) for s in ls)
```

The function `sat` checks if all strings in the list `ls` have unique characters by converting each string to a set, which removes any duplicate characters, and then checking if the length of the set is equal to the length of the original string. The `all` function is used to return `True` if all the conditions inside the generator expression are `True`, and `False` otherwise.
