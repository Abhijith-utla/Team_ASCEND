Here is the solution for this problem:

```python
from typing import List

def sat(ls: List[str]) -> bool:
    return len(ls) == len(set(ls))

# Test
assert sat(["apple", "banana", "apple"])
assert not sat(["apple", "banana", "cherry"])
```

In this solution, we define a function `sat` that checks if the length of the input list is equal to the number of unique elements in the list. The function returns `True` if the list is a set, and `False` otherwise. We use the set data structure to remove duplicates from the list, and then compare the length of the list to the length of the set. If the length of the list and the set are the same, that means that the list contains no duplicate values, and the function returns `True`. Otherwise, the function returns `False`.

We then test this function with two different test cases: one where the list contains no duplicates, and one where the list does contain duplicates. The first test case passes, so the second test case should fail.
