The given function `sat` checks if a list has all unique elements. If the length of the list is equal to the length of the set of its elements, then the list contains all unique elements. If not, the list contains duplicate elements. Therefore, if the length of the list is not equal to the length of the set, then the list does not contain all unique elements.

Here is the Python code that solves the problem:

```python
from typing import List

def sat(ls: List[str]) -> bool:
    return len(ls) == len(set(ls))
```

You can test this function with a list of unique elements:

```python
print(sat(["apple", "banana", "cherry"]))  # True
```

And a list with duplicate elements:

```python
print(sat(["apple", "banana", "cherry", "apple"]))  # False
```
