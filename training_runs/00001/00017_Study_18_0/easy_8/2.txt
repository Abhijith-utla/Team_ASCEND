from typing import List

def sat(ls: List[str]) -> bool:

This function checks if a list has duplicate elements by converting the list to a set. If the length of the set is not equal to the length of the list, it means that there are duplicate elements in the list, so the function returns False. If the lengths are equal, it means that there are no duplicate elements, so the function returns True.
