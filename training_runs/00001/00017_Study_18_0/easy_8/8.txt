Here is the Python code that satisfies the given requirements:

```python
def sat(ls: List[str]) -> bool:
    return len(ls) == len(set(ls))
```

This function checks if all elements in the given list are distinct by converting the list to a set. If the length of the set is equal to the length of the original list, that means all elements were distinct and the function returns True. Otherwise, it returns False. The function is considered to be "solution" (sol) if it satisfies the given conditions.
