Here's the correct solution for the problem:

```python
from typing import List

def sol(ls: List[str]) -> bool:
    return len(ls) == len(set(ls))
```

This function takes a list of strings as input and checks if the length of the list and the length of the set from the list are equal. If they are equal, it means that there are no duplicate elements in the list, so it returns True. If they are not equal, it means that there are duplicate elements in the list, so it returns False.
