Here is the Python solution for the problem:

```python
from typing import List

def sat(ls: List[str]) -> bool:
    return len(ls) == len(set(ls))
```

This function checks if all elements in the list are unique. It does this by converting the list to a set, which by definition cannot contain duplicate elements. If the length of the original list and the set are equal, it means that every element in the list was found exactly once, so the list has no duplicate elements.
