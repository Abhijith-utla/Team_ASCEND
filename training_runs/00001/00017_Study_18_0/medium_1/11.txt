Here is the solution:

```python
from typing import List

def sat(ls: List[str]) -> bool:
    return all(s not in ls for s in ls)

print(sat(["apple", "banana", "cherry"]))  # False
print(sat(["apple", "banana"]))  # True
```

The function `sat` checks if all elements in the list `ls` are not present in the list itself. If all elements are not present, the function returns `False`, otherwise it returns `True`.
