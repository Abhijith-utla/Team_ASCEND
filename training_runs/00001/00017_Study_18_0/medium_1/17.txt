```python
from typing import List

def sat(ls: List[str]) -> bool:
    return all(s not in ls for s in ls)

def sol() -> str:
    return str(sat(['a', 'b', 'c']))
```

It uses the Python built-in function `all` to check if all elements in the list satisfy the condition (not in the list). If all elements are unique, `all` returns `True`; otherwise, it returns `False`.
