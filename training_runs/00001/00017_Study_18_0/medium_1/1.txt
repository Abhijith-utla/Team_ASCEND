Here is a Python solution for the problem. This solution uses Python's built-in set and list operations to solve the problem. It first converts the list of strings to a set, which automatically removes any duplicates. Then it checks if all elements in the set are in the original list, which will only be true if every element is unique.

```python
from typing import List

def sat(ls: List[str]) -> bool:
    return len(ls) == len(set(ls))
```

This function will return True if the list contains no duplicate strings and False otherwise.
