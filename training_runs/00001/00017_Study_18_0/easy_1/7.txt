Here's the solution for the problem. The function `sat` checks if a given list has all unique elements. If the length of the list is equal to the length of the set created from the list, it means that the list has all unique elements.

```python
from typing import List

def sat(ls: List[str]) -> bool:
    return len(ls) == len(set(ls))
```

This function is correct for all inputs of the `sat` function. For instance, if the input list is `['apple', 'orange', 'banana', 'apple']`, the function will return `False` as the length of the set is `4`, and the length of the list is also `4`, but the list still has duplicate elements.
