Here is the Python code that solves the problem:

```python
from typing import List

def sat(ls: List[str]) -> bool:
    return len(ls) == len(set(ls))

print(sat(['apple', 'banana', 'apple']))  # Output: False
print(sat(['apple', 'banana', 'banana']))  # Output: True
```

This function `sat` checks if all elements in the given list are unique. It does this by converting the list to a set, which removes duplicates. If the length of the original list and the set are equal, then all elements are unique, so it returns True. If the lengths are not equal, then the list has at least one duplicate element, so it returns False.
