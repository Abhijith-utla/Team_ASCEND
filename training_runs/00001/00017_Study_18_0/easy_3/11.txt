Here is a Python solution for the problem. The function `sat` checks if all elements in the given list are unique. It uses a set to automatically remove any duplicate elements as sets in Python only contain unique elements. Then it checks if all elements in the list are in the set. If all elements are in the set, it returns `True`, otherwise it returns `False`.

```python
from typing import List

def sat(ls: List[str]) -> bool:
    return all(s not in ls for s in set(ls))
```

The function `sat` is already defined by the user, so you do not need to define it again.

You can check the function with the following code:

```python
def test_sat():
    assert sat(["apple", "banana", "cherry"])
    assert not sat(["apple", "apple", "cherry"])
    assert sat([])
    assert not sat(["apple"])

test_sat()
```

This test function will pass if the function `sat` works correctly.
