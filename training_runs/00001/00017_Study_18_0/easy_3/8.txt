Here is the Python solution:

```python
from typing import List

def sat(ls: List[str]) -> bool:
    return all(s not in ls for s in set(ls))
```

This function checks if all the elements in the list are unique. It does this by converting the list to a set, which removes duplicates and then checks if any element is in the list. If any element is found to be in the list, the function returns False. If the function completes without finding any duplicates, it returns True.
