Here's the solution for the problem:

```python
def sol():
    i = 1000
    while True:
        if sat(i):
            return i
        i += 1
```

This function starts from 1000 and keeps incrementing it until it satisfies the condition `sat(i)`.

The `sat` function is defined as:

```python
def sat(i: int):
    return len(str(i + 1000)) > len(str(i + 1001))
```

This function checks if the length of the string representation of `i + 1000` is greater than the length of the string representation of `i + 1001`. If it is, then `i` is the smallest number that makes both strings of `i + 1000` and `i + 1001` have different lengths.
