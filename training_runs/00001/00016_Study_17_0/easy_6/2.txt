Here is the solution to the problem:

```python
def sol():
    i = 1
    while not sat(i):
        i += 1
    return i

def sat(i: int) -> bool:
    return len(str(i + 1000)) == len(str(i + 10000))
```

The `sat` function checks if the length of the string representation of `i + 1000` is equal to the length of the string representation of `i + 10000`. The `sol` function tries to find such an `i` using a while loop. It continues to increment `i` until `sat(i)` returns `True`.
