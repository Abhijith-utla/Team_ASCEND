Here are the three simpler variants of the original puzzle:

VARIANT1:
```python
def sat(i: int):
    return len(str(i + 1000)) != len(str(i + 1001))
```

VARIANT2:
```python
def sat(i: int):
    return len(str(i + 1000)) == len(str(i + 1001))
```

VARIANT3:
```python
def sat(i: int):
    return len(str(i + 1000)) < len(str(i + 1001))
```
