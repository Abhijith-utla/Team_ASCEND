```python
def sol():
    for i in range(1000):
        if sat(i):
            return i
    return None

def sat(i: int) -> bool:
    return len(str(i + 10)) == len(str(i + 100))

# Testing
assert sat(sol())
```
This program checks all positive integers from 0 to 999 to see if it satisfies the condition (the length of the string representation of a number plus 10 equals the length of the string representation of a number plus 100). If it finds such an integer, it returns it. If it doesn't find such an integer after checking all integers, it returns None.
