Sure, here are three variants of the `sat` function:

VARIANT1:
```python
def sat(x: List[str]) -> bool:
    return all(len(x[i]) <= len(x[i + 1]) and x[i + 1] > x[i] for i in range(len(x) - 1))
```

VARIANT2:
```python
def sat(x: List[int]) -> bool:
    return all(x[i] <= x[i + 1] and x[i + 1] > x[i] for i in range(len(x) - 1))
```

VARIANT3:
```python
def sat(x: List[float]) -> bool:

```
