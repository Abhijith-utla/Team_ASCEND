Here are the three simpler variants of the original puzzle:

VARIANT1:
```python
def sat(x):
    return all(len(x[i]) <= len(x[i + 1]) for i in range(len(x) - 1))
```

VARIANT2:
```python
def sat(x):
    return all(len(x[i]) <= len(x[i + 1]) and x[i + 1] > x[i] for i in range(len(x) - 1))
```

VARIANT3:
```python
def sat(x):
    return all(len(x[i]) <= len(x[i + 1]) for i in range(len(x)))
```
