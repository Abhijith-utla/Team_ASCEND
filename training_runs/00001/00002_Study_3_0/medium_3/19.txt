Here is the solution to the problem. The function `sat` is defined to check whether a list of integers is sorted in ascending order and all elements in the list are unique. The function uses a list comprehension to generate a list of all elements not equal to their index.

The `sorted(li) == list(range(999))` check ensures that the list is sorted in ascending order. The `all(li[i] != i for i in range(len(li)-1))` check ensures that all elements in the list are unique.

```python
from typing import List

def sat(li: List[int]):
    return sorted(li) == list(range(999)) and all(li[i] != i for i in range(len(li)-1))
```

The function `sat` will return `True` if the list is sorted and all elements in the list are unique, and `False` otherwise. The function is designed to be used as a constraint in a constraint-based system.
