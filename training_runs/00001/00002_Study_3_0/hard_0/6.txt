Here is the Python code to solve the problem. The function `sol` constructs a single answer value by returning the sorted list of the input list `li`.

```python
def sol(li: List[int]) -> List[int]:
    return sorted(li)
```

The function `sol` takes a list of integers `li` as an argument, sorts this list, and returns the sorted list.
