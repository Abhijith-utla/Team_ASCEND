Here is the Python solution for the problem:

```python
from typing import List

def sat(li: List[int]) -> bool:
    return sorted(li) == list(range(999)) and all(li[i] == i for i in range(len(li)))
```
This solution checks if the list is sorted in ascending order and if all elements in the list are equal to their indices. The "sorted(li)" part of the function is used to sort the list, "list(range(999))" creates a list of numbers from 0 to 998, and "all(li[i] == i for i in range(len(li)))" checks if all elements in the list are equal to their indices.
