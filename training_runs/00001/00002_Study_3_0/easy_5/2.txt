```python
from typing import List

def sat(li: List[int]) -> bool:
    return sorted(li) == list(range(1, len(li) + 1)) and all(li[i] == i for i in range(len(li)))
```

The function `sorted(li)` sorts the list in ascending order.

The `list(range(1, len(li) + 1))` creates a list of integers from 1 to the length of the list + 1.

The `and` operator is used to check if both conditions are true.
