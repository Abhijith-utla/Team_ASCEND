```python
from typing import List

def sol():
    return sorted([i for i in range(1001)])

def sat(li: List[int]):
    return sorted(li) == list(range(1001))

# Test
assert sat(sol())
```

In the above code, `sol` returns a sorted list of integers from 0 to 1000. The `sat` function checks if the sorted list matches the list of integers from 0 to 1000. The sorted list is obtained by creating a list comprehension that generates a sequence of integers from 0 to 1000 and then sorts this sequence.
