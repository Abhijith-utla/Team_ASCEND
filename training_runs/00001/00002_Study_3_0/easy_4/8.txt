```python
def sat(li: List[int]) -> bool:
    return sorted(li) == list(range(len(li))) and all(li[i] == i for i in range(len(li)))
```
This function checks if a list `li` is sorted in increasing order and all elements in `li` are equal to their corresponding index. The function `sorted(li)` sorts the list `li` in increasing order, and `list(range(len(li)))` generates a list of numbers from 0 to `len(li) - 1`. The `all(li[i] == i for i in range(len(li)))` checks if all elements in `li` are equal to their corresponding index. If both conditions are met, the function returns `True`, otherwise it returns `False`.
