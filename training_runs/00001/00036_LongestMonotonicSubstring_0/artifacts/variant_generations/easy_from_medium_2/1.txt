Here are the three simplified variants for the "sat" function.

VARIANT1:
```python
def sat(x: List[int]):
    return len(x) > 0
```
In this variant, the function only checks if the input list is not empty.

VARIANT2:
```python
def sat(x: List[int], length=13):
    return len(x) >= length
```
In this variant, we are only checking if the input list is not empty.

VARIANT3:
```python
def sat(x: List[int], length=13, s="Dynamic programming solves this puzzle!!!"):
    return len(x) >= length
```
In this variant, the function is checking if the input list is not empty and also checks if the list contains at least one substring in sorted order. But the string s is not being used in the function.
