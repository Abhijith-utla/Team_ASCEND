```python
def sat(x: List[int]):
    return all(x[i] <= x[i + 1] for i in range(len(x) - 1))
```

VARIANT2:
```python
def sat(x: List[int], length=13):
    return all(x[i] <= x[i + 1] for i in range(length - 1))
```
