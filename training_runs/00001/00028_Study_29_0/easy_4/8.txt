# For the test, let's assume that we want to check the list `l` for being a permutation of the range from 0 to 999.
# We can use the `all` function to check if all elements in the list satisfy the condition.
# If all elements satisfy the condition, `all` returns `True`; otherwise, it returns `False`.
# We can use a nested loop to check all possible pairs of elements in the list.
# For each pair of elements, we check if their square difference is less than 10.
# If any pair of elements does not satisfy the condition, we return `False`.
# Otherwise, we return `True`.

def sol():
    l = [i for i in range(1000)]
    return all(i in range(1000) and abs(i * i - j * j) < 10 for i in l for j in l if i != j)
