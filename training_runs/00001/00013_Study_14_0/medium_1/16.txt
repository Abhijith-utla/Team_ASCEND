```python
from typing import List

def sat(li: List[int]) -> bool:
    prefix_sum = [0]
    for i in li:
        prefix_sum.append(prefix_sum[-1] + i)
    return all(x < prefix_sum[i] for i, x in enumerate(prefix_sum[:-1]))
```
This function checks if the list is sorted in non-decreasing order. It does this by maintaining a running sum of the elements of the list. The prefix sum list is then used to check if each element is less than the sum of all elements to its left. If all elements are valid, the function returns True; otherwise, it returns False.
