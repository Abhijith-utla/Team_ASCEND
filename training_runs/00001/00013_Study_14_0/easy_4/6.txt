```python
def sat(li: List[int]) -> bool:
    if not li:
        return False
    smallest = li[0]
    for i in range(len(li)):
        if li[i] < smallest:
            smallest = li[i]
        if li[i] > sum(li) - sum(li[:i]):
            return False
    return True
```

This function works by iterating over the list and comparing each element with the smallest element found so far. If a element is smaller than the smallest one, it updates the smallest element. If an element is greater than the sum of all elements till the current element, it returns False. If the loop finishes without finding a pair where the first element is greater than the sum of all elements till the current element, it returns True.

The function does not perform any operations on the list, so its time complexity is O(n), where n is the length of the list.
