Here is the correct pattern for the Python solution:

```python
def sol():
    def sat(li: List[int]):
        for i in range(1, len(li)):
            if li[i] < li[i-1]:
                return False
        return True

    return sat
```

This solution defines a function `sol` that returns a function `sat`. The `sat` function checks if the list `li` is sorted in ascending order. If the list is not sorted, the function returns `False`, else it returns `True`.

Here is a test case:

```python
def test():
    assert sol()([1, 2, 3, 4, 5])() == True
    assert sol()([1, 2, 3, 4, 6])() == False
    assert sol()([5, 4, 3, 2, 1])() == True

test()
```

Please note that this solution has a time complexity of O(n^2), which is not efficient for large inputs.
