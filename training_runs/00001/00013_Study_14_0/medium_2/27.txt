```python
from typing import List

def sat(li: List[int]):
    for i in set(li):
        if li.index(i) < sum(li[:li.index(i)]):
            return False
    return True
```

This function checks if all elements in the list are less than the sum of all elements before the current element in the list. If any element is less than the sum, it returns False. If the function passes through the entire list without finding any elements that are less than the sum, it returns True.
