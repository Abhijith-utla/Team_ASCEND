```python
def sol():
    class Answer:
        def __init__(self, is_valid):
            self.is_valid = is_valid

    def sat(li: List[int]):
        if len(li) == 0:
            return Answer(True)
        unique_elements = set(li)
        for x in unique_elements:
            if li.index(x) < sum(li[:li.index(x)]):
                return Answer(False)
        return Answer(True)

    answer = sat(li)
    return answer.is_valid
```

This solution first defines a nested function `sat` inside an `Answer` class to handle the answer object. The `sat` function checks if all numbers in the list are less than the sum of the elements before the current element in the list. If any number in the list is greater, the function returns `False` for the answer object. If all numbers are less than or equal to the sum, the function returns `True` for the answer object.
