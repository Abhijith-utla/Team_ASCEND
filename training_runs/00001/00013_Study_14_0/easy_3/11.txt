```python
from typing import List

def sat(li: List[int]) -> bool:
    if not li:
        return True
    total = sum(li)
    current = 0
    for i, num in enumerate(li):
        if current < (total - current) / (len(li) - i) * num:
            return False
        current += num
    return True
```

This function calculates the sum of all numbers in the list and then iterates through the list, checking if the current number is less than the average of the remaining numbers in the list. If it is, it returns False. If it goes through the entire list without finding a number that is greater than the average, it returns True.

The function uses the fact that the sum of all numbers in a list is equal to the sum of the first n numbers (where n is the length of the list) multiplied by the average of the first n numbers, which is equal to the sum of the first n-1 numbers.
