```python
from typing import List

def sat(li: List[int]):
    return all([i <= j for i in li for j in li if i > j])
```

This means that if `li` has any two elements `i` and `j` where `i` > `j`, then `i` must be less than or equal to `j`. The `all` function in Python returns `True` if all the elements in the iterable `li` are true.
