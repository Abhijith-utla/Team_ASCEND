```python
from typing import List

def sat(li: List[int]):
    return all([sum(li[:i]) == i for i in range(20)])
```

This function `sat` checks if all elements in the list `li` are equal to their corresponding index. It does this by summing up all the elements up to the current index (`i`) and checking if the sum is equal to the index.

The function `all` is used to check if all elements in the list satisfy the condition. If all elements satisfy the condition, `all` returns `True`; otherwise, it returns `False`.

The condition in the list comprehension is `sum(li[:i]) == i`, which checks if the sum of all elements up to the `i`-th index is equal to the `i`-th index.

The `range(20)` generates a sequence of numbers from 0 to 19, which are used as indices for the list `li`.
