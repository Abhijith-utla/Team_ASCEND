Here is the Python code for the solution:

```python
def sat(li: List[int]):
    return all([sum(li[:i]) == i for i in range(20)])
```

This function is correct according to the given problem. It returns True if the sum of the first 'i' numbers in the list is equal to 'i' for all 'i' from 0 to 19. If the list is empty, it returns True.
