```python
from typing import List

def sat(li: List[int]):
    for i in range(1, len(li)):
        if li[i] < li[i-1]:
            return False
    return True
```

If it finds such an element, it immediately returns False. If it doesn't find any such element in the list, it returns True, indicating that the list is sorted in ascending order.

The function `sat` assumes that the input list `li` is sorted in ascending order and returns True. If it finds such an element in the list, it returns False.
