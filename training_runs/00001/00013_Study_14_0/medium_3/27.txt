Here is the solution for the problem:

```python
from typing import List

def sat(li: List[int]) -> bool:
    for i in range(len(li)):
        if sum(li[j] for j in range(i+1, len(li))) != li[i]:
            return False
    return True
```

In this solution, for each element in the list, we calculate the sum of elements in the rest of the list starting from the current element. If the sum of these elements is not equal to the current element, we return False as it means our list is not in the correct format. If we finish the loop without finding any mismatch, we return True, indicating that the list is in the correct format.
