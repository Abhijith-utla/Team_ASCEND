Here is the Python solution for the problem:

```python
from typing import List

def sat(li: List[int]) -> bool:
    for i in range(len(li)):
        if sum(li[j] for j in range(i+1, len(li))) != li[i]:
            return False
    return True
```

This solution works by checking each element in the list. It computes the sum of the elements in the rest of the list starting from the current index. If this sum is not equal to the current element, then the function returns False indicating that the input list is not valid according to the definition of the problem. If the function has checked all elements and hasn't returned False, it means that the input list is valid, so it returns True.
