The function `sat` is defined to check if a list `ls` contains the same element twice at different positions, and if the first element is not the same as the second element. Here's how you could define the `sol` function to solve this problem:

```python
def sol():
    def sat(ls, idx1=1, idx2=2):
        return ls[idx1] in ls[idx2] and ls[idx1] != ls[idx2]

    ls = [1, 2, 3, 4, 5]
    for i in range(len(ls)):
        for j in range(i + 1, len(ls)):
            if sat(ls, i, j):
                return [i, j]
    return None
```
