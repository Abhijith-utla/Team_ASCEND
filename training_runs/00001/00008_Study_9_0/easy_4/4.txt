```python
from typing import List

def sat(li: List[str]):
    return all(i.isdigit() for i in li)
```

Each boolean value is True if the corresponding element in the list is a digit, and False otherwise. The built-in `all` function returns True if all the values in the sequence are True, and False otherwise.

The function `isdigit` is a built-in Python method that returns True if all the characters in the string are digits, and False otherwise.
