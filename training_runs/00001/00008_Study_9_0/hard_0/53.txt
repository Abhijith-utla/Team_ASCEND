Here is the Python code for the solution:

```python
def sol():
    return "".join(list("The quick brown fox jumps over the lazy dog")[i] for i in [1, 3, 15, 6, 9, 4, 11, 7, 8, 0, 14, 10, 12, 16, 5, 2, 13])

print(sol())
```
This function returns the sentence "The five boxing wizards jump quickly" based on the indices in the list [1, 3, 15, 6, 9, 4, 11, 7, 8, 0, 14, 10, 12, 16, 5, 2, 13].

The list comprehension generates the corresponding character from the sentence at the indices given by the list and the join function is used to concatenate the characters into a string.
